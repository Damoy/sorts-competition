> Java source structure

	- app:
		Contains different entry points.
		Each app gathers same category comparisons.
		TutorialApp contains a tutorial to who to use our structures.
		
	- core: 
		Contains all algorithms implementations.
		All inherit from the ISort interface.
		
	- execution:
		Contains tools to build scenarios.
		Goal is to ease comparisons.
		
	- generator:
		Contains tools to generate data.
	
	- model:
		Contains our data model
		
	- visualization:
		Contains the sort visualizer.
		Provides sorts comparison graphs.
